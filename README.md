**A phishing email analysis tool that parses `.eml` files and identifies indicators of phishing.**

![Status](https://img.shields.io/badge/status-In%20Progress-yellow)

It takes raw email files and examines their headers, sender information, and authentication results to help determine whether an email is a phishing attempt. It's designed for security analysts and anyone who wants to understand what's really going on behind a suspicious email.
